# Safety App

To run the backend:
Requirements
Node version ^18

Run the following commands
- cd back
- cd functions
- npm run serve
  
if you are having troubles with firebase finding the index in the package.json main to point "lib/src/index.ts"

To run front:

Requires flutter version 3.24.3

Create .env file with the back url "http://127.0.0.1:5001/safety-app-ff42c/us-central1/api" inside the front folder

on ios
- cd front
- cd ios
- flutter run
